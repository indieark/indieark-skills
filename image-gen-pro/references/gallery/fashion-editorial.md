# Fashion Editorial Gallery

> Load when: 需要时尚大片、服装造型、模特摄影、lookbook 或 editorial mood。
> Avoid: 产品包装、技术图、UI 或儿童/敏感人物设定。
> Pairs with: `../methods/photography.md`、`../methods/style.md`、`../methods/composition.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Fashion Editorial`; prompts are rewritten for local use.

## Editorial Look

- Size: `1024x1536`
- Quality: `high`

```text
Create a fashion editorial photograph of [outfit/style].
Model: confident pose, clear garment silhouette, appropriate styling.
Lighting: studio or location light matching the mood.
Composition: full-body or three-quarter crop, clean background relationship.
Avoid: fake brand logos, distorted anatomy, unreadable garment details.
```

## Lookbook Layout

- Size: `1536x1024`
- Quality: `high`

```text
Create a fashion lookbook image for [collection].
Layout: one or more clean poses with consistent styling.
Focus: garment material, cut, layering, accessories.
Mood: [minimal/luxury/street/outdoor/editorial].
Avoid: extra text, busy props, inconsistent outfit logic.
```
