# Isometric Gallery

> Load when: 需要等距场景、城市块、房间、流程空间、游戏建筑或模块化布局。
> Avoid: 人像摄影、海报标题、精确数据图。
> Pairs with: `../methods/composition.md`、`../methods/technical-illustration.md` if later added, `../methods/style.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Isometric`; prompts are rewritten for local use.

## Isometric Room / Space

- Size: `1024x1024`
- Quality: `high`

```text
Create an isometric illustration of [room/space].
Structure: clear floor plane, walls or boundaries, readable object placement.
Details: functional props that explain the use of the space.
Style: clean stylized 3D or vector-like rendering.
Avoid: impossible perspective, clutter, text labels.
```

## Isometric System Map

- Size: `1536x1024`
- Quality: `medium`

```text
Create an isometric system map for [concept].
Modules: distinct zones connected by paths or flows.
Hierarchy: most important module visually dominant.
Palette: coherent, readable, not too saturated.
Avoid: random tiny icons, unclear connections, fake labels.
```
