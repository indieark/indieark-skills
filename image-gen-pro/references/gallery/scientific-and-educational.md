# Scientific & Educational Gallery

> Load when: 需要科普图、教学插图、自然科学、生物、物理或教育视觉。
> Avoid: 精确论文数据图、商业海报或纯艺术图。
> Pairs with: `../methods/infographics.md`、`../methods/research-figures.md`、`../methods/structured-prompts.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Scientific & Educational`; prompts are rewritten for local use.

## Educational Diagram

- Example canvas: `1536x1024`
- Quality: `medium`

```text
Create an educational diagram explaining [topic].
Layout: main visual plus 3-5 callouts.
Style: clean, friendly, classroom-ready.
Labels: short generic labels or blank label areas.
Avoid: fake formulas, invented data, dense paragraphs.
```

## Scientific Illustration

- Example canvas: `1024x1024`
- Quality: `high`

```text
Create a scientific illustration of [subject].
Focus: accurate visible structure, clean background, readable detail.
Rendering: precise but approachable, restrained colors.
Context: include scale or environment only if useful.
Avoid: fantasy anatomy, misleading labels, decorative effects.
```
