# Character Design Gallery

> Load when: 需要角色设定、立绘、头像、服装/道具设定或系列角色一致性。
> Avoid: 没有人物/角色主体，或只需要背景/产品/数据图。
> Pairs with: `../methods/consistency.md`、`../methods/style.md`、`../methods/composition.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Character Design`; prompts are rewritten for local use.

## Character Sheet

- Size: `1536x1024`
- Quality: `high`

```text
Create a character design sheet for [character].
Views: front view plus small detail callouts for outfit/accessories.
Identity anchors: age range, silhouette, hairstyle, clothing, signature colors.
Style: clean concept art, readable design, neutral background.
Avoid: multiple unrelated characters, fake notes, inconsistent costume details.
```

## Portrait Concept

- Size: `1024x1536`
- Quality: `high`

```text
Create a portrait concept of [character].
Framing: bust or half-body, expressive face, distinctive silhouette.
Details: clothing materials, accessories, posture, emotion.
Lighting: supports personality and genre.
Avoid: overdecorated costume, hidden face, extra text.
```
