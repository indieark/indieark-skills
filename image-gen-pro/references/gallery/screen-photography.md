# Screen Photography Gallery

> Load when: 需要屏幕中的产品、设备展示、电脑/手机使用场景或 app-on-device 摄影。
> Avoid: 真实 UI 实现、精确可读小字或无设备场景的普通 UI mockup。
> Pairs with: `../methods/photography.md`、`../methods/ui-mockups.md`、`../methods/product.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Screen Photography`; prompts are rewritten for local use.

## Device In Context

- Example canvas: `1536x1024`
- Quality: `high`

```text
Create a realistic photograph of [device] showing [app/content type] on screen.
Scene: believable desk or hand-held context, clean props.
Screen: visible interface blocks but no tiny readable text.
Lighting: screen glow balanced with environment light.
Avoid: fake brand marks, unreadable dense UI, distorted device shape.
```

## Product Demo Screen

- Example canvas: `1536x1024`
- Quality: `high`

```text
Create a product demo photograph featuring [software/product] on a screen.
Composition: device angled to show screen and surrounding workflow context.
UI: simplified dashboard or app layout with placeholder text blocks.
Mood: professional, clear, not a marketing hero card.
Avoid: exact credentials, real private data, random logos.
```
