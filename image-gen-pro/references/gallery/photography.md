# Photography Gallery

> Load when: 需要真实摄影、生活方式照片、人物/物体实拍感、镜头和光线参考。
> Avoid: 插画、UI、信息图或强风格化绘画。
> Pairs with: `../methods/photography.md`、`../methods/composition.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Photography`; prompts are rewritten for local use.

## Patterns

### Natural Lifestyle Photo

- Use when: 人物、商品或场景需要可信生活化照片。
- Size: `1536x1024`
- Quality: `high`
- Prompt skeleton:

```text
Create a natural lifestyle photograph of [subject] in [environment].
Camera: medium shot, eye-level or slightly candid angle.
Lighting: natural available light, soft shadows.
Details: believable props, realistic surfaces, subtle imperfections.
Color: natural editorial grade.
Avoid: staged stock-photo look, fake text, over-smoothed skin or surfaces.
```

### Studio Portrait / Object

- Use when: 头像、人物半身、物体或产品需要干净棚拍。
- Size: `1024x1536`
- Quality: `high`
- Prompt skeleton:

```text
Create a studio photograph of [subject].
Framing: [close-up/half-body/full object], clean silhouette.
Lighting: soft key light, gentle fill, optional rim light.
Backdrop: simple neutral background.
Focus: sharp on [important feature].
Avoid: busy background, extra accessories, exaggerated retouching.
```

### Documentary Scene

- Use when: 想要纪实、街拍、工作现场或真实环境感。
- Size: `1536x1024`
- Quality: `medium` or `high`
- Prompt skeleton:

```text
Create a documentary-style photograph of [scene].
Moment: [action or situation] feels observed, not posed.
Camera: handheld realism, natural perspective.
Environment: specific location details, restrained clutter.
Light: available light matching the time and place.
Avoid: cinematic over-staging, fake signage, impossible camera angles.
```

### Macro Detail

- Use when: 需要材质、纹理、珠宝、植物、食物或产品细节。
- Size: `1024x1024`
- Quality: `high`
- Prompt skeleton:

```text
Create a macro photograph of [detail].
Focus: crisp detail on [feature], shallow depth of field.
Texture: describe material, grain, moisture, fibers or reflections.
Lighting: controlled highlights, soft falloff.
Avoid: abstract blur, extra objects, unrealistic scale.
```
