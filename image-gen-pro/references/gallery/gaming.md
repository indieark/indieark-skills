> Avoid: 非游戏商业产品、真实摄影或论文图。

Source: adapted from `gpt_image_2_skill` gallery category `Gaming`; prompts are rewritten for local use.

## Game Key Art

- Size: `1536x1024`
- Quality: `high`

```text
Create game key art for [genre/theme].
Hero: [character/object/location] dominates with a strong silhouette.
Composition: cinematic foreground, readable midground, atmospheric background.
Mood: [danger/cozy/mystery/comedy/epic], with clear genre signal.
Avoid: fake title text, crowded tiny enemies, unclear main subject.
```

## Game Asset Icon

- Size: `1024x1024`
- Quality: `medium`

```text
Create a game asset icon for [item/skill].
Shape: instantly readable at small size, centered, clean outline.
Material: [metal/magic/wood/fire/ice/etc.] with controlled highlights.
Background: simple contrast shape or transparent-ready flat background.
Avoid: tiny decorations, text labels, overly complex scene.
```
