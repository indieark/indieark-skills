# Anime & Manga Gallery

> Load when: 需要动画、漫画、轻小说封面、赛璐璐、分镜或角色图风格。
> Avoid: 用户要求真实摄影、产品图、技术图或严肃论文图。
> Pairs with: `../methods/style.md`、`../methods/consistency.md`、`../methods/composition.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Anime & Manga`; prompts are rewritten for local use.

## Character Key Visual

- Example canvas: `1024x1536`
- Quality: `high`

```text
Create an anime key visual of [character].
Character: clear silhouette, expressive pose, distinctive outfit and accessories.
Composition: full-body or half-body, readable background, strong focal point.
Style: clean line art, controlled cel shading, polished color palette.
Avoid: extra characters, fake text, cluttered effects over the face.
```

## Manga Panel Mood

- Example canvas: `1536x1024`
- Quality: `medium`

```text
Create a manga-style scene showing [moment].
Panel language: dramatic framing, clear action line, expressive emotion.
Rendering: monochrome ink, screentone texture, crisp contour lines.
Composition: foreground subject with readable background context.
Avoid: unreadable speech bubbles, random captions, confusing action direction.
```
