# Technical Illustration Gallery

> Load when: 需要产品剖面、机械示意、硬件图、工程插图或操作说明。
> Avoid: 纯艺术海报、自然摄影或品牌 mood board。
> Pairs with: `../methods/research-figures.md`、`../methods/infographics.md`、`../methods/structured-prompts.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Technical Illustration`; prompts are rewritten for local use.

## Exploded View

- Example canvas: `1536x1024`
- Quality: `high`

```text
Create a technical exploded view of [object].
Structure: separated parts aligned along clear axes.
Style: precise linework, subtle shading, neutral background.
Labels: blank callout areas or short generic labels.
Avoid: impossible parts, fake serial text, decorative clutter.
```

## Instruction Diagram

- Example canvas: `1536x1024`
- Quality: `medium`

```text
Create a technical instruction diagram for [operation].
Layout: 3-5 clear steps with consistent icons or panels.
Arrows: simple directional arrows only.
Style: clean industrial manual illustration.
Avoid: dense paragraphs, unsafe actions, ambiguous tool placement.
```
