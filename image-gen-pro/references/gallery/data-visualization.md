# Data Visualization Gallery

> Load when: 需要数据可视化 mockup、图表海报、dashboard chart、抽象数据视觉。
> Avoid: 需要真实数值精确绘图；那应使用代码/表格工具生成真实图表。
> Pairs with: `../methods/infographics.md`、`../methods/research-figures.md`、`../methods/ui-mockups.md`。

Source: adapted from `gpt_image_2_skill` gallery category `Data Visualization`; prompts are rewritten for local use.

## Abstract Data Poster

- Example canvas: `1536x1024`
- Quality: `medium`

```text
Create a data visualization poster about [topic].
Visual: abstract chart-like shapes, clear hierarchy, no real numeric claims.
Layout: title-safe area, chart area, legend placeholder.
Style: clean editorial data art.
Avoid: fake precise values, unreadable labels, misleading statistics.
```

## Dashboard Chart Mockup

- Example canvas: `1536x1024`
- Quality: `medium`

```text
Create dashboard chart mockups for [domain].
Components: line chart, bar chart, summary cards, filter controls.
Data: believable placeholder patterns, not exact real numbers.
Style: clear axes areas and legends, compact UI.
Avoid: tiny unreadable labels, random spikes, decorative chart junk.
```
