# image-gen-pro Skill

This is the canonical installable Skill package.

It contains:

- `SKILL.md` - mandatory image generation / image editing trigger and workflow instructions.
- `scripts/` - `imagen` CLI with planning, dry-run, route execution, history, media, and local job commands.
- `references/` - CLI, workflow, provider contract, API, gallery, methodology, and safety references.
- `agents/` - UI metadata.

It keeps provider API details, gallery patterns, and methodology in references instead of the top-level Skill body.
