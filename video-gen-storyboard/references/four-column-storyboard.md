# Four-Column Storyboard Layout

> Load when: 已理解 `storyboard-board.md` 的故事板母图理念，并且短剧、广告、产品宣传或系列内容适合用四栏版式统一主视觉、细节、场景和可执行分镜。
> Avoid: 还没有决定故事板母图的信息密度和区域主次；先读 `storyboard-board.md`。
> Pairs with: `storyboard-board.md` 提供上层概念；`../../video-gen-assets/references/setting-director.md` 产出项目设定；`../../video-gen-assets/references/character-sheet.md` / `../../video-gen-assets/references/scene-sheet.md` 提供参考；`../../video-gen-assets/references/seedance-handoff.md` 登记和确认。

四栏是故事板母图的一种项目级版式，不是故事板本身的定义。它不是营销海报，而是把项目核心视觉和可执行视频节拍压缩到同一张参考母图。四栏是默认参考结构，不是硬模板；可以按项目把栏目改成更贴合的“主视觉 / 角色关系 / 场景冲突 / 镜头节拍”等结构，也可以升级为更完整的制作设定板。

## When to Use

Use when:

- 任务是短剧、广告、产品宣传、系列内容或团队确认稿。
- 用户需要先看“这个项目长什么样”，再决定是否生成视频。
- 角色、产品、场景和风格都需要统一。
- 第四栏需要承载可执行的镜头节拍；默认可用 15 秒九宫格爆点分镜，也可以按场景改成其他数量的关键镜头。

Avoid when:

- 用户只有一个简单动作，且明确要快速生成。
- 只需要单段镜头控制，不需要项目总览。
- 角色或场景设定还没稳定，先回到人物/场景设定图。
- 核心效果主要是烟尘、爆炸、粒子、魔法能量、混乱群体或强抽象视觉，且无法被清楚栏目和镜头节拍表达；这类任务优先走 prompt-heavy 或混合路径。

## Layout Contract

| Column | Purpose | Must Contain |
| ------ | ------- | ------------ |
| 1 | 主视觉 / 封面定调 | 角色或产品的统一形象，项目第一印象 |
| 2 | 细节 / 视觉锤 | 人设细节、产品细节、道具、手部、表情、材质 |
| 3 | 剧情场景 / 使用场景 | 情绪曲线、空间关系、产品使用或冲突场景 |
| 4 | 镜头节拍 / 3x3 15 秒分镜 | 连续关键镜头，决定最终视频镜头顺序 |

前三栏负责稳定角色、产品、场景和风格；第四栏负责让 Seedance 按可执行镜头节拍运动化。默认写成 3x3，但如果当前项目更适合 4、6、8 或更多镜头，按实际节奏改写。

## Confirm Before Generation

生成四栏或自定义项目故事板前，先确认：

```text
项目故事板确认：
- 项目类型：短剧 / 广告 / 产品宣传 / 系列内容
- 故事板结构：默认四栏 / 自定义栏目
- 第一栏用途：
- 第二栏用途：
- 第三栏用途：
- 第四栏或分镜区的实际镜头数量：
- 角色/产品锁定规则：
- 场景和风格锁定规则：
- 用户要先看项目总览，还是直接进入某一段视频：
- 图片规格：4K 或更高
- 图内文字：栏目标签 / 镜头表 / 对白与声音说明是否需要进入图中
- 视频提示词策略：短读图规则 / 需要补充复杂特效文字
```

用户确认后再生成故事板图。故事板图生成后，必须再确认第四栏或分镜区是否真的能指导视频生成；不能只确认“画面好看”。

## Short Drama Storyboard Prompt

```text
参考用户创意和已确认的角色/场景设定，生成一张 4K 专业 AI 短剧前期视觉故事板。
比例 16:9，横向四栏排版，每栏独立，有细边框分隔。
整体风格：<题材/画幅/主色/光线/材质/镜头语言>。
允许清晰栏目标签、镜头表、对白、声音和情绪说明；文字必须可读、图文一致。禁止杂乱拼贴、自由散乱排版、水印、乱码文字、无意义伪字。

第一栏【短剧主视觉 / 封面定调】：
展示 <短剧名称> 的核心海报级画面，<主角形象> 占据主视觉，<关键冲突或危险/爽点> 清楚可见。

第二栏【人设细节 / 视觉锤】：
3 个微距或局部特写，上下排列。
细节 1：<眼神/表情/身份识别点>。
细节 2：<服装/道具/材质/伤痕/配饰>。
细节 3：<反派线索/场景光影/关键物件>。

第三栏【剧情场景 / 情绪曲线】：
3-4 个剧情场景，上下排列。
场景 1：<冲突建立>。
场景 2：<危险逼近或关系转折>。
场景 3：<高光动作或情绪爆发>。
场景 4：<悬念定格或下段入口>。

第四栏【15 秒爆点分镜 / 镜头节拍】：
默认 3x3 排列，镜头 1 到镜头 9 连续叙事；如当前场景更适合 <镜头数量> 个关键镜头，可按实际节奏合并或扩展：
1 <黄金钩子>；2 <人设亮相>；3 <冲突升级>；4 <细节特写>；
5 <核心反转>；6 <视觉高潮>；7 <情绪共鸣>；8 <记忆强化>；9 <悬念定格>。

短剧信息：
标题：<title>
题材：<genre>
核心看点：<hook>
目标观众：<audience>
核心场景：<scene>
情绪基调：<emotion arc>
主色调：<palette>
完播钩子：<ending hook>

整体要求：四栏风格统一，角色外观一致，服装、道具、场景和光线连续。
可以放入清晰可读的栏目标签、镜头表和必要对白/声音提示；不要加入和视频生成无关的装饰性文字。
```

## Product / Ad Storyboard Prompt

```text
根据用户广告创意、产品信息和可选参考图，生成一张 4K 专业产品宣传片前期故事板。
比例 16:9，横向四栏排版，每栏独立，有细边框分隔。
真实高级商业广告质感，<品牌主色>，背景干净，无水印，无乱码文字。允许清晰产品信息、镜头表和声音/情绪短说明，但文字必须可读且服务视频生成。

第一栏【产品主视觉】：
展示 <产品名称> 的完整外观，产品占据画面主要面积，材质、颜色、轮廓和标志性结构稳定。

第二栏【产品细节】：
3 个微距特写，上下排列。
细节 1：<核心功能或显示区域>。
细节 2：<材质、结构或包装>。
细节 3：<使用瞬间或质感高光>。

第三栏【使用场景 / 目标用户】：
3-4 个真实使用场景，上下排列。
场景 1：<用户痛点或开场环境>。
场景 2：<产品介入>。
场景 3：<核心卖点被看见>。
场景 4：<情绪或结果>。

第四栏【宣传片分镜 / 镜头节拍】：
镜头 1 <开场氛围>；镜头 2 <用户痛点>；镜头 3 <产品登场>；
镜头 4 <产品细节>；镜头 5 <使用动作>；镜头 6 <核心卖点>；
镜头 7 <情绪变化>；镜头 8 <产品高光>；镜头 9 <结尾定格>。

产品信息：
产品名称：<name>
产品类型：<type>
核心卖点：<benefit>
目标用户：<audience>
使用场景：<use cases>
宣传片情绪：<emotion>
品牌主色：<palette>

禁止：不要让产品形状变化，不要让包装颜色变化，不要自由拼贴，不要画面杂乱，不要乱码文字。
```

## Seedance Video Prompt Template

```text
参考 @故事板图片 生成一条 <时长> 秒 <画幅> 视频。
这是本条视频的主参考图，请优先读取图内栏目、镜头表和视觉说明。

要求：
1. 按照故事板第四栏的实际分镜顺序生成视频；若使用九宫格，则按从左到右、从上到下读取。
2. 保持故事板中的人物/产品、服装/外观、场景、光线、主色调和整体风格一致。
3. 前三栏用于稳定主视觉、细节和场景，第四栏用于镜头顺序；不要把默认四栏或九格当成不可调整的限制。
4. 可以加入轻微环境音效或产品音效，但不要喧宾夺主。

禁止：
不要出现乱码文字。
不要让人物手部或产品形状变形。
不要让画面杂乱。
不要新增故事板之外的无关角色、场景、卖点或剧情结果。

不要逐字复述故事板绘图提示词；按图内既定栏目和镜头表执行。
```

## Quality Checklist

- 是否是 16:9 横向四栏结构，而不是自由拼贴。
- 第一栏是否能立刻说明主视觉或产品主视觉。
- 第二栏是否有清晰的人设细节、产品细节或视觉锤。
- 第三栏是否能说明剧情场景、使用场景和情绪曲线。
- 第四栏是否包含连续可执行分镜，默认 3x3 但可按场景调整。
- 人物、服装、产品、场景和色调是否统一。
- 是否是 4K 或更高规格，图内栏目、镜头表、对白或声音说明是否清晰可读。
- 视频 prompt 是否保持简短，说明前三栏和第四栏各自用途，而不是全文复述故事板绘图提示词。

## Common Failures

| Symptom | Fix |
| ------- | --- |
| 变成海报，没有执行分镜 | 强化第四栏镜头节拍，按实际镜头数量写清顺序 |
| 四栏风格不一致 | 回到 style bible，重复主色、光线、材质 |
| 产品变形 | 增加产品锁定规则和细节栏 |
| 短剧角色漂移 | 先补 `../../video-gen-assets/references/character-sheet.md`，再重做故事板 |
| 视频只看主视觉不看分镜 | 在 `video-prompt.txt` 明确第四栏决定镜头顺序 |

## Registration

生成四栏故事板图片后，进入 `../../video-gen-assets/references/seedance-handoff.md`：

```bash
videogen storyboard add <project-id> --storyboard-id <id> --image <storyboard.png> --prompt <storyboard-prompt.txt> --video-prompt <video-prompt.txt>
```
