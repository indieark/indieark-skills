# Storyboard Method

> Load when: 需要 2-4 个可见节拍，短剧、广告、产品展示或社媒钩子需要递进。
> Avoid: 用户明确只要单个连续镜头、克制氛围或严格素材控场；改用 `clapperboard.md`。
> Pairs with: `../scenes.md` 的 Product / Drama / Social 场景；`../director.md` 的动作节奏字段。

分镜方法把一次生成拆成少量连续画面节拍，适合短剧、广告、产品展示和有起承转合的社媒短视频。它和场记板互斥，一次任务只选一种主结构。

## When to Use

Use when:

- 用户需要多个视觉节拍。
- 用户要短剧、广告、产品展示或开头反差。
- 需要把 5-15 秒拆成 2-4 个阶段。
- 卖点/情绪/记忆点需要分阶段递进，单镜头放不下。

Avoid when:

- 用户明确只要一个连续镜头。
- 用户指定安静、克制、单一动作的场景。
- 分镜会让任务超出 Seedance 2.0 的短视频时长（<= 15 秒）。
- 用户需要严格的素材/参数控场（优先 `clapperboard.md`）。

## Beat Fields

只选必要字段，不要机械全填。每个 Beat 维持在一行内可读。

```text
Beat：编号或时间段。
画面：主体动作 + 关键变化（不写静态形容词）。
镜头：景别 / 运镜 / 速度（与上一拍要有差异或递进）。
声音：音乐节拍 / 音效 / 对白；无声场景写"延续静默"。
钩子：本拍要让观众记住的一个点（每条视频建议只有 1 次主钩子）。
```

## Beat Pattern

短视频默认 3 拍模板，按时长可压缩为 2 拍或扩到 4 拍：

```text
0-2 秒：建立主体/场景/钩子。
2-5 秒：主体动作或卖点展开。
5-8 秒：转折、结果或记忆点。
```

## Prompt Pattern

```text
分镜：
Beat 1 (0-2s)：画面... / 镜头... / 声音... / 钩子...
Beat 2 (2-5s)：画面... / 镜头... / 声音...
Beat 3 (5-8s)：画面... / 镜头... / 声音... / 记忆点...

生成指令：
按上述分镜节拍生成一个 ... 秒 ... 画幅视频，每拍之间使用平滑过渡，
不要剪开镜头；保持主体一致，不要加入无关文字或 logo。
```

## Example

User asks:

```text
帮我做一个 8 秒社媒短视频，主推一杯冷萃咖啡的清凉感，9:16，要有钩子和记忆点。
```

Good storyboard:

```text
Beat 1 (0-2s)：画面：玻璃杯落入桌面，冰块跳动溅起细密水雾。镜头：低角度特写，固定。声音：轻冰块碰撞 + 起拍鼓点。钩子：第 1 秒就听到清脆冰声。
Beat 2 (2-5s)：画面：黑色咖啡液从壶口缓慢倒入杯中，冲过冰块形成乳白翻涌。镜头：缓慢推近至杯口，浅景深。声音：水流持续 + 鼓点稳态。
Beat 3 (5-8s)：画面：水雾在杯壁凝结成水珠缓慢下滑，最后镜头微抬看到品牌色背景。镜头：从特写慢慢拉远到中近景。声音：低频环境音收束，鼓点淡出。记忆点：水珠下滑 + 品牌色。
```

## Self Check

- 每个节拍是否可见（动作/光线/位置至少有一项变化）。
- 是否只保留 1 个核心事件或核心卖点。
- 节拍之间是否能平滑过渡，没有硬切。
- 是否没有把简单场景拆得过碎（2 秒以内的拍子要谨慎）。
- 是否和 `clapperboard.md` 冲突；如果用户要单镜头控场，优先场记板。
- 是否把所有声音都写"生成声音"——应明确节拍/音乐/音效角色。
