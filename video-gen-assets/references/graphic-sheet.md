# Graphic Asset Prompt Method 素材方法

> Load when: 需要生成或登记可复用的视觉素材（logo、图标、纹理、贴纸、转场元素、风格底图、UI/字幕样式件等），用作参考图输入或后期叠加。
> Avoid: 需要锁定的是人物、空间或具体道具（去对应的人物/场景/道具卡）；只要一次成品图且不复用。
> Pairs with: `principles.md` 讲 reference 纯净度与三层链路；`character-sheet.md` / `scene-sheet.md` / `prop-sheet.md` 是其他四类资产；`seedance-handoff.md` 登记和引用。

素材是**不绑定单一角色、空间或具体道具**的通用视觉元件。它的判别轴不是视图角度，而是**复用范围**：这件素材要复用多远——一次性、项目内、还是跨项目沉淀成系列风格 master。素材有两种用法：作为视频模型的**参考图输入**（风格底图、纹理、构图参考），或作为**后期叠加件**（logo、水印、角标、转场贴纸、字幕/标题样式件）。

纯度规则按用途分流（细则见 `principles.md`）：作为视频模型参考图输入的素材遵循 reference 纯净度，零语义文字；作为后期叠加的素材（logo、字幕样式件、带字角标）本就是要带字的成品件，按成品规格制作，不与参考图输入混用。

## 形态档位（轴 = 复用范围）

素材的梯度轴是**复用范围**，从轻到重（概念 SSOT 见 `../../video-gen-pro/references/concepts.md`）：

| 档位 | 形态 | 复用范围 | 选档判据 |
| ---- | ---- | -------- | -------- |
| L0 轻 | 一次性单件 | 本镜/本段一次性使用 | 只用一次、不进项目库长期复用 |
| L1 中 | 项目套件 | 本项目内反复复用的一组 | 项目内多镜/多段统一使用（同一套图标、纹理、角标、转场） |
| L2 重 | 跨项目系列风格 master | 跨项目沉淀的品牌/系列风格件 | 品牌 logo、系列统一字幕样式、可跨项目复用的风格底图 master |

L0 也可以直接随镜头处理、不进库；只有需要被反复调用时才登记为 L1/L2 素材资产。

## Required Inputs

- 素材名和用途（参考图输入 / 后期叠加；具体做什么）。
- 类别：logo / 图标 / 纹理 / 贴纸 / 角标 / 转场元素 / 风格底图 / 字幕样式件。
- 复用范围：一次性 / 项目内 / 跨项目。
- 视觉规格：尺寸、比例、是否需要透明背景（PNG alpha）。
- 配色、风格关键词，与项目风格 bible 的关系。
- 若作参考图输入：明确"给视频模型当输入"，遵循纯净度；若作后期叠加：明确成品规格和是否带字。

## Confirm Before Generation

```text
素材确认：
- 素材：
- 用途：参考图输入（风格/纹理/构图） / 后期叠加（logo/角标/转场/字幕件）
- 复用范围：一次性 / 项目套件 / 跨项目 master
- 类别：
- 视觉规格：尺寸 / 比例 / 是否透明背景
- 配色和风格：
- 纯度：参考图输入零文字 / 后期叠加按成品件
```

用户确认后再生成或登记。

## 提示词与制作要点

- 作参考图输入时：纯图像，写清风格、配色、纹理质感和构图意图，不写图内文字。
- 作后期叠加件时：按成品规格（含透明背景、精确尺寸、可带字），交由图片生成 Skill 产出；这类带字成品件不作为视频模型参考图输入。
- 透明背景 PNG、批量套件、图生图等需求转交图片生成 Skill（见全局 `image-gen-pro`），本文件只负责定位、复用判断和登记。

## Registration

生成或拿到素材后：

```bash
videogen asset add <project-id> --type graphic --name <素材名> --file <image>
```

登记时 `purpose` 写清"参考图输入风格底图""项目角标套件""跨项目品牌 logo master"等用途和复用范围。外部已有素材用 `--source external` 登记同一来源，不复制文件。
