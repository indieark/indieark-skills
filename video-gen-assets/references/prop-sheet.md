# Prop Sheet Prompt Method

> Load when: 需要生成或修正道具/产品设定图，用于后续镜头里道具外观、比例和材质的一致性。
> Avoid: 道具只是场景里不重要的陈设（并入场景卡即可）、用户只要成品静态图、或已有稳定道具图。
> Pairs with: `scene-sheet.md` 提供道具所在空间；`character-sheet.md` 处理角色随身道具的佩戴状态；`seedance-handoff.md` 登记和引用。

道具卡用于锁定一件**反复出镜、需要保持外观一致**的物件（关键道具、产品、武器、载具、招牌物等）。它让后续故事板和视频段都能复用同一张道具锚点，而不是每镜重画。随身小配饰随角色卡走，不必单独立卡；只有当道具有独立戏份、需要特写或跨镜头跨段复用时才做道具卡。

本文件的硬规则：道具卡直接作为视频模型的参考图输入，必须**纯图像、零文字**。图内不写任何标题、品名、字段表、尺寸标注或文字标签；图内文字会被视频模型当成要还原的画面元素，污染成片。品名、材质、尺寸等设定写进提示词正文和资产登记，不画进图里。纯几何测量线和对齐参考线不算文字。

## 形态档位（从轻到重）

道具资产是一条从轻到重的三档梯度（概念 SSOT 见 `../../video-gen-pro/references/concepts.md`）：

| 档位 | 形态 | 说明 | 选档判据 |
| ---- | ---- | ---- | -------- |
| L0 轻 | 纯净底单图 | 单角度道具图，纯色/纯净底或带必要环境背景；画幅按道具外形取正方形或长方形均可，不强制特定比例 | 一次性、单角度出镜、复用低；只需锁一个主外观 |
| L1 中 | 主图 + 右 1/3 顶/侧拆解 | 左 2/3 主视图，右 1/3 上半顶视图、下半侧视图或局部特写 | 跨镜头复用、需要多角度对齐、有可动部件或开合状态 |
| L2 重 | 多视图道具卡（加底视图） | 正/侧/顶/底多视图 + 材质与关键结构特写，master | 主线关键道具、产品 master、系列复用、强一致性需求 |

不要默认最轻档，也不要无脑上最重档：单角度一次性用 L0，跨镜头多角度用 L1，关键 master 才用 L2。

这条梯度决定一件道具**做**多重；至于实际**喂**视频模型时挑哪张（本镜一次性出镜的道具给单图即可，不必把多视图道具卡全塞进去），见 `../../video-gen-cinematography/references/input/image-input.md` 的 Input Economy。

## Required Inputs

- 道具名和剧情功能（它在故事里做什么、谁在用）。
- 类别：手持道具 / 产品 / 武器 / 载具 / 招牌物 / 机关。
- 整体造型、比例和相对尺寸参照（相对手掌、人体或场景）。
- 材质与工艺：金属/木/塑料/玻璃/织物，表面处理、磨损、做旧。
- 主色、辅色和关键图案、logo、铭文位置（作为画面元素，不是图内文字标签）。
- 可动部件或状态：开/合、亮/灭、满/空、完好/损坏。
- 项目统一风格。

缺少材质、比例或关键识别点时，先补齐或问用户，再生成。

## Confirm Before Generation

生成或调用外部图像工具前，先给用户确认一份短包：

```text
道具设定图确认：
- 道具：
- 用途：关键道具锁定 / 产品 master / 武器设定 / 多视图 master
- 设定图类型：single（纯净底单图）/ main-plus-detail（主图+拆解）/ multi-view（多视图道具卡）
- 输出尺寸：至少 2K；材质细节丰富、产品 master 或大量复用优先 4K
- 整体造型和比例：
- 材质和工艺：
- 主色/辅色/图案：
- 可动部件或状态：
- 一致性要求：
```

用户确认后再生成图片。结果不符合造型、材质、比例或识别点时，先修道具设定图，再进入场景或故事板。

## 提示词格式

```text
道具设定图提示词：
- 画面布局：<single 纯净底单图 / main-plus-detail 左 2/3 主视图右 1/3 上顶下侧 / multi-view 正侧顶底多视图加材质特写>，干净纯色或浅灰背景。纯图像，不写任何标题、品名、字段表或文字标签。
- 纯图像理由：这张图作为视频模型参考图输入，图内文字会被还原成画面元素并污染成片。品名、材质、尺寸写进提示词正文，不画进图里。
- 道具描述：<道具名（不写进图）>，<类别>，<整体造型>，<比例与尺寸参照>。
- 材质工艺：<材质>，<表面处理/磨损/做旧>，<反光/透明/粗糙等质感>。
- 颜色图案：<主色>，<辅色>，<图案/logo/铭文位置，作为画面元素绘制>。
- 结构状态：<可动部件>，<开合/亮灭/满空/完损等需要展示的状态>。
- 对齐参考：可保留浅灰水平/垂直测量参考线用于多视图比例对齐，这是几何辅助线不是文字。
- 输出尺寸：分辨率最低 2K；multi-view 多视图道具卡等需要横版多视图对齐时 16:9 不低于 2560x1440，single 纯净底单图等配套图按道具外形取正方形或长方形均可、不强制比例；材质细节丰富、产品 master 或大量复用优先 4K，推荐 3840x2160。
- 一致性要求：同一道具，造型、材质、颜色、图案、比例在所有视图中完全一致；纯图像、无文字，适合后续故事板、CUT 图和 Seedance 视频参考。
```

## Prop Continuity Tokens

生成视频 prompt 或故事板 prompt 时，重复这些锚点：

```text
道具锚点：<整体造型>、<主色和材质>、<关键图案/logo>、<相对尺寸>、<固定识别点>。
状态锚点：<开合/亮灭/满空/损耗状态> 从上一镜延续。
```

视频模型需要反复读取可见特征；写 prompt 时重复道具锚点和状态锚点，并标明哪张道具卡作为参考图输入。

## Failure Diagnosis

| Symptom | Fix |
| ------- | --- |
| 道具变形 | 增加多视图道具卡，写清整体造型和比例 |
| 材质漂移 | 固定材质、表面处理和反光质感 |
| 比例失真 | 写相对尺寸参照（相对手掌/人体/场景） |
| 图案错乱 | 固定图案/logo 的位置、颜色和大小 |
| 图内出现文字 | 删除图内品名/标注，设定只写进提示词正文 |

## Registration

生成完成后：

```bash
videogen asset add <project-id> --type prop --name <道具名> --file <image>
```

登记时 `purpose` 写清“关键道具锁定”“产品 master”“多视图 master”等用途。
