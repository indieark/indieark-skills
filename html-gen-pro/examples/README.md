# Examples

跑通的真实产物长期收录。**不是教程**，是给 Agent 在新会话里看「过去做过的好作品」的对照基准——防止每次重新临场发挥。

## 收录标准

一条 example 进入这里必须满足：

1. 已经走完完整 S0-S5 状态机（不是半成品 / 试跑）。
2. S5 五维 critique 至少 4 维 Pass，**第 5 维「差异化记忆点」必须 Pass**。
3. 反 AI slop 黑名单 0 命中。
4. 用户已肉眼验证过（浏览器打开过且未要求大改）。

不达标的产物留在 `_work/html_runs/` 即可，不进 examples/。

## 目录结构

每条 example 一个子目录，含：

- `index.html` + `style.css`（+ 任何附属资源）— 真实产物
- `README.md` — 决策记录：S3 锁定的美学方向 / S4 选择 / S5 critique 摘要 / 残余风险
